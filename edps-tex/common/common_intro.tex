This document describes how to reduce \instname\ data with
the \edpsgui\ (Graphic User Interface), the dashboard  of the ESO Data
Processing System (\edps), which is the recommended interface to reduce
data from ESO telescopes.
Details on the \instname\ data reduction stream and how to configure
the reduction to meet specific scientific needs are also given.


For a more extensive documentation on the \edpsgui\ itself, consult
the dedicated
manual \href{https://edps.readthedocs.io/en/latest/edpsgui/index.html}{here}.

For a description of the \instname\ pipeline itself, consult the pipeline manual available at:\\
\href{https://www.eso.org/sci/software/pipe_aem_table.html}{https://www.eso.org/sci/software/pipe\_aem\_table.html}.

Note: this tutorial refers to:
\begin{itemize}
\item \instname\ instrument pipeline named \pipename,
version \pipelinevers.
\item \instname\ workflow: \wkfname
\item \edps\ version \edpsversion.
\item \edpsgui\ version \edpsguiversion.
\end{itemize}
