
\subsubsection{Apply the same parameters to multiple datasets and configurations}

A common user case is to apply the same set of optimized parameters to
other datasets or other configurations. This is possible with the {\bf
  Other configurations} tab of the configuration editor.
To do so, proceed as follows:
\begin{itemize}
\item From the Reduction Queue tab, select the dataset/configuration
  that has the parameters you want to adopt for other
  reductions.
  \item Open the configuration editor by clicking on the weel-button
    on the right-hand side, at the end of the line of the selected
    configuration.
  \item If necessary, edit the parameters.
  \item On the {\bf Other configurations} tabs, select the datasets or
    configurations you want to reduce with the same parameters of the
    ``Current configuration''. Only datasets/configurations present in
    the Reduction Queue can be selected.
  \item Go on the Parameters tab of the configuration editor and click
    on the button {\bf Copy to selected configurations}. Selected
    configurations that were labelled with ``NEW'' will be updated,
    selected configurations that were labelled with
    ``COMPLETED/FAILED/ABORTED'' will not be updated, but ``NEW''
    configurations will be created instead.
  \item If you have edited paramters on the 3rd step, please note that
    the original ``Current configuration'' will not be updated by
    pressing the button {\bf Copy to selected configurations}; only
    those selected in the ``Other configurations'' tab will be
    updated. If you want the edited parameters to be copied also to
    the ``Current configuration'', then press the ``SAVE'' button
    (available only if the Current configuration is labelled as NEW)
    or ``Create new configuration'' button (in this case the Current
    configuration is not updated, but a ``NEW'' configuration is
    created instead).
  
\end{itemize}

The newly created or updated configurations are now ready for
reduction.
