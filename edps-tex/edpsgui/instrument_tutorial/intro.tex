\section{Introduction}

\subsection{Scope}

This document describes in details the \edpsgui\ dashboard (Graphic
User Interface), the recommended infrastructure to reduce data from
the ESO telescope, using the ESO Data Processing System
(\edps). Each instrument has a dedicated tutorial that guides the user
through the data reduction for that specific instrument.

Note: this manual refers to:
\begin{itemize}
\item \edps\ version \edpsversion.
\item \edpsgui\ version \edpsguiversion.
\end{itemize}
